== Description ==

決済プラットフォーム「Stripe」の決済ボタンを設置できるプラグインです。
<a href="https://s-page.biz/ssc/">https://s-page.biz/ssc/</a>

== Installation ==

このプラグインは、WordPressダッシュボードから直接インストールできます。

1. [プラグイン]メニューに移動し、[新規追加]をクリックします。
2. "Simple Stripe Checkout" を検索します。
3. "Simple Stripe Checkout" プラグインの横にある "今すぐインストール" をクリックします。
4. プラグインをアクティブにします。
5. [Simple Stripe Checkout] メニュー > [初期設定] で、Stripeの [公開キー] と [シークレットキー] を設定します。
6. [Simple Stripe Checkout] メニュー > [新規登録] で、商品の [商品名] [価格] [通貨] [提供者] [ボタンの名前] を登録します。
7. [Simple Stripe Checkout] メニュー > [商品一覧] で、[ショートコード] をコピーします。
8. 任意の投稿、固定ページに [ショートコード] をペーストします。
